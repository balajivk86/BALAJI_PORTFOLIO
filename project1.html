EVENT MATRIX - Smart Event Management System

A full-stack event booking system with QR ticketing and organizer dashboard.

Project Overview

This system allows users to browse events, register, and receive QR-based tickets. Organizers can manage events, track registrations, and monitor analytics using a dashboard.

My Role

I designed and developed the complete system including UI/UX design, backend logic, database integration, QR code generation, and email automation.

Tools & Technologies

  • Figma – UI/UX design and prototyping
  • HTML, CSS, JavaScript – Frontend development
  • Python (Flask) – Backend logic
  • Google sheets – Database
  • QR Code Generation
  • Email Integration (SMTP)

Key Features

  • QR Code Ticket Generation
  • Event Registration System
  • Organizer Dashboard
  • Email Confirmation
  • Staff management
